# Sanural Backend - Async API & Excel Parser Integration

Монолитный Enterprise-каркас бэкенда на Python/Django для интернет-магазина Sanural. Проект полностью укомплектован для работы в изолированных Docker-контейнерах.

## 🛠️ Реализованная архитектура спринта:
1. **`models.py`** — Спроектированы таблицы базы данных PostgreSQL. Реализовано динамическое поле `second_api_key` для бесшовного хранения второго API-хэша без хардкода.
2. **`admin.py`** — Настроена кастомная панель управления. Добавлена инлайн-модификация цен и остатков сантехники (`list_editable`) напрямую из общей таблицы.
3. **`api_manager.py`** — Разработан асинхронный сетевой контроллер на базе `httpx`. Реализованы защитные тайм-боксы в 10 секунд для предотвращения дедлоков СУБД.
4. **`excel_parser.py`** — Написан скоростной обработчик прайс-листов на `openpyxl` с использованием транзакционного метода `update_or_create`. В логику парсера заложена автоматическая наценка в 25% на розницу.

## 🚀 Инфраструктурный деплой:
Проект разворачивается в один клик через Docker-compose. Веб-сервер переведён на высокопроизводительный асинхронный рантайм **Uvicorn** (ASGI).
